## 한국어

### 지원 및 문의

KaninDex 사용 중 문제가 발생했거나 앱스토어, 기능, 개인정보, 음악 사용 권리, 제휴 및 후원 관련 문의가 있다면 아래 이메일로 연락해 주세요.

지원 이메일: [shteosis@gmail.com](mailto:shteosis@gmail.com)

문의 시 다음 정보를 함께 보내주시면 문제 확인에 도움이 됩니다.

- 사용 중인 macOS 버전
- KaninDex 앱 버전
- 선택한 Dex 출신 또는 LLM 제공자 유형
- 로컬 LLM 사용 여부 및 사용 중인 모델 이름
- 문제가 발생한 화면 또는 작업 단계
- 가능하다면 오류 메시지 또는 재현 순서

### 원격 LLM 관련 안내

Codex OAuth, 외부 API, 클라우드 LLM 제공자를 사용하는 경우 네트워크 연결, 제공자 인증 상태, API 키, 모델 사용 가능 여부, 제공자 서버 상태에 따라 기능이 실패할 수 있습니다. 민감한 프로젝트에서는 원격 제공자 사용 전 개인정보 처리방침과 해당 제공자 정책을 확인해 주세요.

### 개인정보 처리방침

개인정보 처리방침은 다음 URL에서 확인할 수 있습니다.

## English

### Support and Contact

If you experience an issue with KaninDex or have questions about the App Store listing, features, privacy, music permissions, partnership, or sponsorship, please contact us by email.

Support email: [shteosis@gmail.com](mailto:shteosis@gmail.com)

Including the following details can help us investigate the issue.

- macOS version
- KaninDex app version
- Selected Dex commissioning track or LLM provider type
- Whether a local LLM is being used and the model name
- Screen or operation step where the issue occurred
- Error message or reproduction steps, if available

### Remote LLM Notice

When Codex OAuth, an external API, or a cloud LLM provider is used, features may fail depending on network connectivity, provider authentication status, API keys, model availability, or provider service status. For sensitive projects, please review the privacy policy and the selected provider's policy before using a remote provider.

### Privacy Policy

The privacy policy is available at the following URL.

[https://facta-leopard.github.io/KaninDex/privacy/](https://facta-leopard.github.io/KaninDex/privacy/)

## Deutsch

### Support und Kontakt

Wenn bei der Nutzung von KaninDex ein Problem auftritt oder wenn Sie Fragen zum App-Store-Eintrag, zu Funktionen, Datenschutz, Musikrechten, Partnerschaften oder Sponsoring haben, kontaktieren Sie uns bitte per E-Mail.

Support-E-Mail: [shteosis@gmail.com](mailto:shteosis@gmail.com)

Die folgenden Angaben helfen bei der Untersuchung des Problems.

- macOS-Version
- KaninDex-App-Version
- Ausgewaehlter Dex-Ernennungsweg oder LLM-Anbietertyp
- Ob ein lokales LLM verwendet wird und der Modellname
- Bildschirm oder Arbeitsschritt, bei dem das Problem aufgetreten ist
- Fehlermeldung oder Schritte zur Reproduktion, falls verfuegbar

### Hinweis zu Remote-LLMs

Wenn Codex OAuth, eine externe API oder ein Cloud-LLM-Anbieter verwendet wird, koennen Funktionen je nach Netzwerkverbindung, Authentifizierungsstatus des Anbieters, API-Schluessel, Modellverfuegbarkeit oder Dienststatus des Anbieters fehlschlagen. Bei sensiblen Projekten sollten Sie vor der Verwendung eines Remote-Anbieters die Datenschutzerklaerung und die Richtlinie des ausgewaehlten Anbieters pruefen.

### Datenschutzerklaerung

Die Datenschutzerklaerung ist unter folgender URL verfuegbar.
